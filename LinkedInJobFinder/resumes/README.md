# Resumes

Drop your resume here. Supported formats: `.pdf`, `.docx`, `.md`, `.tex`, `.txt`.

If you have multiple versions (e.g., one tailored for backend roles, one for ML), drop them all — the tool reads every file in this folder and uses the union of titles + skills.

You can also drop a `search_profile.md` here to express what you actually *want* (vs. what your resume says you *can* do): must-haves, deal-breakers, salary floor, interest areas. The fit-scoring step uses both.

Everything in this folder except this README is gitignored.
